<a name="installation"></a>
## Installation
```
Download Zip
Inclue LimepayApi.cs
Include Directories
/Models
/Resources
/Exceptions
from MerchantApi in your Project

using LimepayApi.Net.Exceptions;
using LimepayApi.Net.Helpers;
using LimepayApi.Net.Resources;
using LimepayApi.Net.Models;
```
